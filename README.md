## 🛠️ Integrated Architecture & Workflow

### **Landing Page** → **Dashboard** → **AI Assistant Side Panel**

```
Landing Page
   │
(Authenticate + Personalize)
   ▼
Dashboard (Personal Workspace)
   │
(Real-time Data + Widgets + Customization)
   ▼
AI Side Panel (Contextual Help, Workflow Automation)
   │
(Multi-AI Responses, Aggregate, Format, Return)
   ▼
User/View/Automate/Update across all components
```

### **How Components Work Together:**
- **Landing Page**: AI-tailored visuals and features hook users with personalized content
- **Dashboard**: Provides control, customization, and smart productivity—all real-time and adaptive
- **AI Side Panel**: Universal assistant understanding dashboard context, suggesting and automating actions
- **Results Integration**: Side panel outputs instantly update dashboard widgets and personalize future experiences

## 🚀 Getting Started

### **1. Clone the Repository**
```bash
git clone https://github.com/bishalpokhrel1/future-stick-dashboard.git
cd future-stick-dashboard
```

### **2. Install Dependencies**
```bash
# Install Node.js dependencies
npm install

# Or use yarn
yarn install
```

### **3. Environment Setup**
```bash
# Copy environment template
cp .env.example .env

# Configure your API keys and settings
# - AI service API keys (OpenAI, Google, etc.)
# - OAuth credentials
# - Database connections
```

### **4. Development Server**
```bash
# Start development server
npm run dev

# Or use Docker for full stack
docker-compose up -d
```

### **5. Access the Platform**
- **Landing Page**: http://localhost:3000
- **Dashboard**: http://localhost:3000/dashboard
- **API Documentation**: http://localhost:3000/docs
